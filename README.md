* <kbd>Server Monitoring App</kbd> A fully deployed, real-time telemetry tracking application. Designed to aggregate server health metrics, monitor uptime across distributed nodes, and provide actionable alert orchestration for bare-metal and self-hosted environments.

* <kbd>[Windows Debloat Toolkit](https://github.com/The-IT-Guy/Windows-11-Debloat)</kbd> An automated PowerShell scripting framework designed to aggressively strip out telemetry, disable tracking hooks, remove bloatware, and optimize performance across Windows corporate environments.
  
* <kbd>Silhouette OS</kbd> A custom, specialized Linux distribution built on a hardened base, focusing entirely on native system hardening, bare-metal efficiency, and structural privacy.

* <kbd>Tungsten / Flow Theme</kbd> Custom KDE Plasma 6 global UI environments, featuring ground-up designs for color schemes, SDDM login screens, and custom KWin bash scripts.

* <kbd>GroceryCompare</kbd> A private custom utility application optimized for mobile workflows, utilizing direct UPC code scanning to parse, track, and contrast price indexes.

* <kbd>Restaurant Roulette</kbd> A mobile decision-engine application utilizing custom randomization algorithms and location data to filter and select dining options based on user-defined parameters.

I have engineered a unified **AI Stack**—orchestrating <kbd>Claude</kbd>, <kbd>ChatGPT</kbd>, and <kbd>Gemini</kbd> to work collaboratively directly through **CLI environments**. Rather than using them in isolation, I direct a hierarchy of models, passing context and code between them to leverage their specific computational strengths. 

* **High-Context Thought Leadership:** Utilizing Claude as an analytical, "Senior" engineering thought partner for overarching system architecture and complex problem-solving.
* **Rapid Code Production:** Directing ChatGPT and Gemini via command-line logic pipelines for rapid script generation, deep refactoring, and infrastructure blueprinting.
* **Cross-Model Validation:** Feeding the output of one model into another to validate logic, parse errors, and debug self-hosted deployments securely directly within the shell.
